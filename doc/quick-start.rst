Quick Start
===========

Installation
------------

::

   pip install dataiter

You'll need Python 3.6 or later.

Usage
-----

Similar to NumPy, Pandas etc. we'll typically import the module under a
shorter alias and then access everything via that alias.

::

   import dataiter as di
